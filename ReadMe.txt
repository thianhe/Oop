操作（鍵盤）—— “W”：向上行走      “↑”：向上射擊
             “S”：向下行走     “↓”：向下射擊
           “A”：向左行走      “←”：向左射擊
           “D”：向右行走      “→”：向右射擊

秘技（鍵盤）——“Q” ：切換武器       “R”：回半血
            “N” ：跳至下一層     “M”：加一塊錢
            “G” ：受傷（扣半血） “B”：跳至Boss房間
            “H” ：扣剩半血       “C”：清除玩家目前所在房間之所有怪物
